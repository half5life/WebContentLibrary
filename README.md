# WebContentLibrary

Это старый проект на базе ASP.NET MVC, предназначенный для управления каталогом контента и файлов.

### Описание
Проект представляет собой веб-приложение для каталогизации элементов с поддержкой различных расширений файлов. Разработан в учебных целях.

### Стек технологий
- **Платформа:** .NET Framework (ASP.NET MVC 5)
- **Доступ к данным:** Entity Framework 6
- **Аутентификация:** Microsoft ASP.NET Identity
- **Интерфейс:** Bootstrap 3, jQuery
